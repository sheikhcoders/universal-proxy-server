# Contributing to Universal Proxy Server

Thank you for your interest in contributing! We welcome contributions from the community.

## How to Contribute

### 1. Fork the Repository

Fork the repository on GitHub and clone your fork locally:

```bash
git clone https://github.com/YOUR_USERNAME/universal-proxy-server.git
cd universal-proxy-server
```

### 2. Set Up Development Environment

```bash
# Install dependencies
npm install

# Create a feature branch
git checkout -b feature/amazing-feature

# Make your changes
# ...

# Test your changes
npm run build
npm start
```

### 3. Development Guidelines

- **Code Style**: Follow the existing TypeScript style
- **Type Safety**: Maintain strict TypeScript types
- **Testing**: Add tests for new features
- **Documentation**: Update README.md for user-facing changes
- **Commits**: Use conventional commit messages

### 4. Submit a Pull Request

1. Push your changes to your fork
2. Create a Pull Request against the `main` branch
3. Describe your changes and their purpose
4. Link any related issues

## Development Commands

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev

# Build for production
npm run build

# Run tests
npm test

# Lint code
npm run lint

# Format code
npm run format
```
